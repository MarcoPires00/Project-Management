# Google Sheets Dashboard Structure

## Dashboard Overview
The dashboard is designed to provide an overview of the project's status, covering various metrics important for project management.

## Columns
The following columns are included in the dashboard:

1. **Project Name**: The name of the project.
2. **Start Date**: The date when the project commenced.
3. **End Date**: The projected completion date of the project.
4. **Status**: Current status of the project (e.g., In Progress, Completed, On Hold).
5. **Budget**: Total budget allocated for the project.
6. **Spent**: Money spent to date.
7. **Team Members**: Names of team members involved in the project.
8. **Milestones**: Key milestones with dates achieved.
9. **Risks**: Identified risks associated with the project.

## Metrics
The following metrics are tracked:

- **Completion Rate**: Percentage of completed tasks versus total tasks.
- **Budget Utilization**: Ratio of spent budget to total budget allocated.
- **Timeline Adherence**: Assessment of time management with actual versus planned timelines.

## Data Layout
The data layout is structured to ensure clarity and ease of access to information. Each row corresponds to a distinct project with all metrics being updated weekly.
